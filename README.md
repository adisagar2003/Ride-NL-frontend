# RideNL

<ul>
<li>Finding nearby drivers within 2km radius across main areas of St. John's, NL</li>
<li>3d interactivity</li>
<li>JWT Authentication</li>
</ul>

<h1>Frontend Tools Used </h1>
1. react-three-fiber
2. Tailwind CSS
3. react-toastify
4. Axios
5. Redux persist

<h1>Backend Tools Used</h1>
1. Sendgrid
2. JWT
3. Mongoose/MongoDB
4. Express
5. NodeJS
